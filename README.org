* Scrap and publish COVID-19 data

This repository scraps COVID-19 data [[https://www.santepubliquefrance.fr/maladies-et-traumatismes/maladies-et-infections-respiratoires/infection-a-coronavirus/articles/infection-au-nouveau-coronavirus-sars-cov-2-covid-19-france-et-monde][from Sant√© Publique France]] daily
updates then publishes accumulated data on [[https://www.data.gouv.fr/fr/admin/dataset/5e689ada634f4177317e4820/][data.gouv.fr]].

* Prerequisit

This runs on the JVM 8 or 11.

It requires =curl= and a =DATAGOUV_API_KEY= environment variable.
